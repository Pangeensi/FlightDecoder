# FlightDecode
基于FlightAware和WireShare的ADS-B本地解码项目。

程序通过提取指定样本文件获取数据，然后完成对数据的解码保存等工作。

样本同工程仓库一并保存，在使用时需要单独提取并保存在指定路径。

样本文件路径路径：D:/FlightAwareData/sample

# 运行环境：Visual Studio 2019及其以上。
